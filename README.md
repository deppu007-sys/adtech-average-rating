# ⭐ AdTech Average Star Rating System

## 📌 Project Overview
This project calculates average star rating for ads/products and helps analyze performance using rating distribution.

## 🚀 Features
- Add ratings (1 to 5 stars)
- Calculate average rating
- Show rating distribution
- Simple AdTech simulation

## 🧠 Use Case
Used in AdTech platforms to improve:
- CTR (Click Through Rate)
- Conversion Rate
- Trust Score

## 💻 Tech Used
- Python

## 📊 Example Output
Average Rating: 4.0  
Distribution: {1:1, 2:1, 3:1, 4:3, 5:4}
